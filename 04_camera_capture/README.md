# 04 Camera Capture

This demo creates an Isaac Sim camera and reads frames directly in Python.

What it teaches:

- how to place a camera in the scene
- how to retrieve `RGBA` frames from the camera object
- how to inspect camera metadata such as intrinsics when available

Why it matters for later demos:

- `09` will subscribe to the ROS 2 image topic that comes from this same style of camera
- `12-15` will reuse the same camera output pattern as VLA visual input

Run:

```powershell
python.bat C:\Users\Administrator\Documents\isaac_sim学习\04_camera_capture\demo.py
```
